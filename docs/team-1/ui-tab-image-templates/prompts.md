# UI Tab Image Templates

Stand: 2026-06-17

Diese Prompts beschreiben die erzeugten Apple-like Vorlagebilder fuer die
Kommunikations-UI. Gemeinsamer Stil fuer alle Tabs:

- 16:9 Desktop-App-Mockup, Light Mode
- Hintergrund `#F6F3F5`, Text `#100A19`
- Akzent `#5A58A6`, Sekundaer `#888BBF`
- Statusfarben: Erfolg `#6BCF7F`, Warnung `#FFD700`, Fehler `#FF6B6B`
- Ruhiger Apple-like Productivity-Stil, 8px Radius, feine Linien, subtile
  Glasflaechen, keine Marketing-Heroes, keine dekorativen Blobs

## Dashboard

High-fidelity UI mockup fuer den Dashboard-Tab von "Kommunikation": linke
Sidebar, kompakter Header, Dashboard-Grid mit Modulstatus fuer API,
Datenbank, Matrix, LLM und User-Dummy, aktuelle Gruppen, neue Feed-Eintraege,
Agent-Feed und Knowledge-Graph-Zusammenfassung.

## Gruppen

High-fidelity UI mockup fuer den Gruppen-Tab: Master-Detail-Layout mit
Gruppenliste, Suche, Plus-Aktion, aktiver Gruppe "Projekt Phoenix",
Beschreibung, Mitglieder mit Dummy-User-Avataren, verknuepftem Matrix-Raum,
Tags und letzter Aktivitaet.

## Matrix-Chat

High-fidelity UI mockup fuer den Chat-Tab: linke Sidebar, sekundare
Gruppen-/Channel-Liste, zentraler Chatverlauf, professionell kompakte
Nachrichten, rechter Kontextbereich mit Gruppeninfo und AI-Hinweisen,
Composer mit Attach- und Send-Icon, Status "Matrix verbunden".

## Wiki

High-fidelity UI mockup fuer Wiki / Knowledge Base: Artikelliste mit Suche
und Tags, zentraler Lesebereich, rechte Metadaten-Spalte mit Autor, Gruppe,
verknuepften Themen und Entscheidungen. Wirkt wie eine premium Team-Notes-App.

## Vernetzungswolke

High-fidelity UI mockup fuer Knowledge Graph / Vernetzungswolke: grosser
Graph-Canvas mit Personen, Gruppen, Themen, Entscheidungen und Wiki-Artikeln
als klar unterscheidbare Nodes, feine Beziehungslinien, Filterleiste und
rechter Inspector fuer den ausgewaehlten Node.

## AI Feed / LLM-Agent

High-fidelity UI mockup fuer AI Feed / LLM-Agent: Fokus auf rechtem AI-Panel
oder eigenem AI-Tab, Feed-Elemente, Zusammenfassung, Aufgabenliste mit
Checkboxen, Priorisierung, Feedback-Buttons mit Daumen hoch/runter und
Status "LLM Mock/Fallback aktiv".
